# Changelog

All notable changes to this project are documented in this file.

## [1.1.0] - 2026-02-14

- Added shared generator utilities in `graphene_shared.py` used by both scripts.
- Added common CLI options to generators: `--output-dir`, `--force`, `--quiet`, and `--profile`.
- Added regression tests in `tests/test_generators.py`.
- Added GitHub Actions CI workflow to run tests on push and pull request.
- Added runtime dependency management with `requirements.txt`.
- Updated documentation for usage, testing, and citation.

## [1.0.0] - 2025-02-06

- Initial public release of Martini 3 graphene structure/topology generators.
